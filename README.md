#### 快速上手
```shell script
npm install weui-icon
```

npm模块下的 `weui-icon` 目录如下：
```text
- styles
    - filled // filled 是 表示该系列图标的样式
        - add.css // add 图标的 css
        - add.svg // add 图标的 svg
        - ...
    - outlined // 结构和 filled 一致，区别是图标的样式
    - weui-icon.css // 所有图标的 css 合集
```

图标样式可见: https://bearj.github.io/icon/dist/
